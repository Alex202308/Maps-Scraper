# 🍷 Maps Scraper — сбор данных об алкомаркетах с Яндекс.Карт и 2ГИС

Проект для автоматического сбора информации о магазинах алкогольной продукции с двух популярных картографических сервисов, слияния результатов и получения сводной таблицы с усреднённым рейтингом.

## 📦 Возможности

- **Парсинг Яндекс.Карт** – извлечение названий, адресов и рейтингов из поисковой выдачи.
- **Парсинг 2ГИС** – обход карточек организаций, переход на вкладку «Отзывы» и получение точного рейтинга.
- **Мультипроцессность** – одновременная обработка нескольких городов.
- **Слияние данных** – сопоставление магазинов по названию и адресу с помощью нечёткого сравнения, устранение дубликатов, расчёт среднего рейтинга.
- **Автоматическое создание папок** – результаты сохраняются в структуру `Город/сервис/`, объединённые файлы в `merged/Город/`.
- **Защита от перезаписи** – при повторном запуске создаётся файл с временной меткой.

## 🛠 Требования

- Python 3.10+
- Установленный [Google Chrome](https://www.google.com/chrome/)
- [ChromeDriver](https://chromedriver.chromium.org/) (Selenium управляет им автоматически через `webdriver-manager` или встроенные средства)
